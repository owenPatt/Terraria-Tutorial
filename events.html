What are the Lunar Events

The Lunar Events are time-based events that end after a certain amount of time has passed. These events drop powerful and class specific items, making players utilize every minute of these events.

Blood Moon

The blood moon is the only pre-hardmode solar event. This event does not drop any important items, however, it provide a way for the players to collect a good amount of money early on into the game. The blood moon does not have any special enemies that spawn, instead many zombies and flying eyes spawn.

Solar Eclipse

The solar eclipse is post-mechanical bosses. This event spawns randomly whenever it turns day-time after the mechanical bosses have been defeated. This event spawns all new enemies and drops all new items for the players. Most of these items are only good for the warrior classes, however, there are some execptions. The main item that is dropped from this event is the Broken Hero Sword. This is used to make most late game warrior items, so it is very important for progession for that class.

Pumpkin Moon

The Pumpkin moon is spawned by the player using the Pumpkin Moon Medallion at any point of the night, however, it should be used at the beginning of night to give the most time to complete it. This event, unlike the blood moon and solar eclipse, comes in waves. Each wave being harder than the last. If you are able to survive long enough, bosses will begin to spawn. These bosses drop important loot for the summoner classes. Other class items will drop, however, this is a very important event for the summoner classes to take part in.

Frost Moon

The Frost Moon is spawned by the player using the Naughty Present at any point of the night, however, it should be used at the beginning of night to give the most time to complete it. This event, unlike the blood moon and solar eclipse, comes in waves. Each wave being harder than the last. If you are able to survive long enough, bosses will begin to spawn. These bosses drop important loot for the Mage and Ranger classes. Other class items will drop, however, this is a very important event for the these two classes to take part in.
